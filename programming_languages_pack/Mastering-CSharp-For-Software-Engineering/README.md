# 🚀 Mastering C# for Software Engineering (Beginner → Professional)

Welcome to the **Mastering C# for Software Engineering** full course repository.  
This repository contains **all lesson notes, source code, assignments, mini projects, and real-world projects** created during the YouTube course.

---

## 🎓 Course Information
- **Course Name:** Mastering C# for Software Engineering  
- **Level:** Beginner → Advanced → Professional  
- **Technology Stack:** C#, .NET 8, SQL Server, EF Core, ASP.NET Core, Web API  
- **Platform:** YouTube + GitHub Learning  
- **Instructor:** Daham Sathmina (ODSDLABS)

---

## 🎯 Course Goals
By the end of this course, you will be able to:

✅ Write clean and scalable C# code  
✅ Understand OOP concepts deeply  
✅ Build Console, Desktop, Web, and API applications  
✅ Work with SQL databases using ADO.NET and EF Core  
✅ Apply SOLID principles and Design Patterns  
✅ Build real-world projects for your portfolio  

---

## 📌 Course Roadmap (Phases)

| Phase | Title | Key Topics |
|------|-------|------------|
| 01 | Foundations | Syntax, Variables, Conditions, Loops, Methods |
| 02 | OOP | Classes, Inheritance, Polymorphism, Collections |
| 03 | Advanced C# | LINQ, File Handling, Async/Await, SOLID |
| 04 | Databases | SQL, ADO.NET, Entity Framework Core |
| 05 | .NET Apps | Windows Forms/WPF, MVC, Web API, JWT |
| 06 | Pro Skills | GitHub, Clean Code, Design Patterns |
