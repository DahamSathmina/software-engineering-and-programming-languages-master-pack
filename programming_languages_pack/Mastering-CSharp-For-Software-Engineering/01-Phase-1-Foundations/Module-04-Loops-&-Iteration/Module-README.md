# Module 04: Loops & Iteration in C#
## Mastering C# for Software Engineering (Beginner to Professional)

---

# Overview

In this module, you will learn how to repeat tasks efficiently using loops.

Loops are one of the most important concepts in programming because they help you:
- Avoid repeating code
- Automate repetitive tasks
- Build real-world applications like menus, games, and systems

---

# Learning Objectives

By the end of this module, you will be able to:

- Understand how loops work in C#
- Use `for`, `while`, and `do-while` loops
- Apply `break` and `continue`
- Work with nested loops
- Build loop-based real-world programs
