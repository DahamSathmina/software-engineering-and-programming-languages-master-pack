# Objective

This assignment will help you practice loops in C# and understand how to:

- Control iterations using break and continue
- Solve real-world programming problems

1. What is a loop in programming?
2. What is the difference between for loop and while loop?
3. What is the purpose of do-while loop?
4. What does the break statement do?
5. What does the continue statement do?

# Part 2: Practical Tasks

---

## Task 1: Print Numbers 1 to 20

Write a program using a loop to print numbers from 1 to 20.

---

## Task 2: Even Numbers

Write a program to print all even numbers between 1 and 50.

---

## Task 3: Sum of Numbers

Write a program that calculates the sum of numbers from 1 to 100.

Example output:
```bash
Sum = 5050
```
---

## Task 4: Multiplication Table

Create a program that:
- Takes a number
- Prints its multiplication table (1 to 10)

Example:
```bash
5 x 1 = 5
5 x 2 = 10
...
```
---

## Task 5: Break Statement

Create a loop that:
- Prints numbers from 1 to 10
- Stops when number reaches 7

---

## Task 6: Continue Statement

Create a loop that:
- Prints numbers from 1 to 10
- Skips number 5

---

## Task 7 (Challenge): Guessing Game

Create a simple number guessing game:
- Set a secret number (example: 7)
- User keeps guessing until correct
- Use loop to repeat input
