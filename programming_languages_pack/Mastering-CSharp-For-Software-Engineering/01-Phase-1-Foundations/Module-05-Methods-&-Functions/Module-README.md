# Module 05: Methods & Functions in C#
## Mastering C# for Software Engineering (Beginner to Professional)

---

# Overview

In this module, you will learn how to organize your code using **methods (functions)**.

Methods help you:
- Avoid repeating code
- Make programs modular
- Improve readability and reusability
- Build professional-level applications

---

# Learning Objectives

By the end of this module, you will be able to:

- Understand what methods are
- Create and call methods in C#
- Use parameters and return values
- Apply method overloading
- Understand recursion basics
