- Prints your name
- Is called from `Main()`

## Task 2: Parameter Method

- Prints a greeting message

```bash

```

## Task 3: Add Two Numbers

- Takes two integers
- Returns their sum

## Task 4: Calculator Methods

- Subtract

- Divide

Test both methods.
